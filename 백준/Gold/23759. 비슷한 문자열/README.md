### 문제 설명

<p>길이가 같은 문자열 $S$와 $T$가 비슷한 문자열이라는 것은 다음과 같이 정의된다.</p>

<ul>
	<li>$1 \leq i \leq \vert S \vert$을 만족하는 $i$에 대해, $S[i] = T[i]$인 $i$가 존재한다.</li>
</ul>

<p>길이가 $K$인 문자열 원소 $N$개로 이루어진 배열이 있다. 해당 배열에서 순서를 바꾸지 않고 0개 이상의 원소를 지워 모든 인접한 원소가 비슷한 문자열이 되도록 하고 싶다. 최소 몇 개의 원소를 지워야하는지 구해보자.</p>

### 입력 

 <p>첫째 줄에 정수 $N$, $K$가 공백을 사이에 두고 주어진다. $(1 \leq N \leq 5 \times 10^5, 1 \leq K \leq 10)$</p>

<p>둘째 줄부터 $N$개의 줄에 걸쳐 길이가 $K$인 문자열이 한 줄에 하나씩 주어진다.</p>

<p>모든 문자열은 알파벳 소문자로만 구성되어 있다.</p>

### 출력 

 <p>첫째 줄에 정답을 출력한다.</p>
