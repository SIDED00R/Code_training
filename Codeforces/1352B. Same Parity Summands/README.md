## 문제 설명

두 개의 양의 정수 $$$n$$$ ($$$1 \le n \le 10^9$$$)과 $$$k$$$ ($$$1 \le k \le 100$$$)이 주어집니다. 숫자 $$$n$$$을 같은 홀짝성을 가진 $$$k$$$개의 양의 정수의 합으로 표현해야 합니다 (즉, $$$2$$$로 나누었을 때 같은 나머지를 가집니다). 다시 말해, 모든 $$$a_i \gt 0$$$이고 $$$n = a_1 + a_2 + \ldots + a_k$$$이며, 모든 $$$a_i$$$가 동시에 짝수이거나 홀수인 $$$a_1, a_2, \ldots, a_k$$$를 찾아야 합니다. 만약 그러한 표현이 존재하지 않는다면, 이를 보고해야 합니다.

## 입력

첫 번째 줄에는 정수 $t$ ($1 \le t \le 1000$)가 포함되어 있으며, 이는 입력의 테스트 케이스 수를 나타냅니다. 다음으로, $t$개의 테스트 케이스가 주어지며, 각 테스트 케이스는 한 줄에 하나씩 나옵니다. 각 테스트 케이스는 두 개의 양의 정수 $n$ ($1 \le n \le 10^9$)과 $k$ ($1 \le k \le 100$)로 구성됩니다.

## 출력

각 테스트 케이스에 대해 다음을 출력하세요: 답이 존재하는 경우 $$$a_i$$$를 포함하여 YES를 출력하고 (여러 개의 답이 있을 경우 그 중 하나를 출력하세요); 답이 존재하지 않는 경우 NO를 출력하세요. 단어 "YES"와 "NO"의 글자는 대소문자 구분 없이 출력할 수 있습니다.
