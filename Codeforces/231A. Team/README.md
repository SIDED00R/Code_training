## 문제 설명

어느 날 세 명의 절친인 페티야, 바샤와 토냐는 팀을 결성하고 프로그래밍 대회에 참가하기로 결정했습니다. 참가자들은 보통 프로그래밍 대회에서 여러 문제를 제시받습니다. 대회 시작 훨씬 이전에 친구들은 최소 두 명이 해결책에 확신이 있을 경우에만 문제를 구현하기로 결정했습니다. 그렇지 않으면 친구들은 문제의 해결책을 작성하지 않을 것입니다. 이 대회는 참가자들에게 n 개의 문제를 제공합니다. 각 문제에 대해 누가 해결책에 확신이 있는지 알고 있습니다. 친구들이 해결책을 작성할 문제의 수를 찾는 것을 도와주세요.

## 입력

첫 번째 입력 줄에는 단일 정수 n (1 ≤ n ≤ 1000)이 포함되어 있으며, 이는 대회의 문제 수를 나타냅니다. 그 다음 n 줄에는 각각 세 개의 정수가 포함되어 있으며, 각 정수는 0 또는 1입니다. 만약 줄의 첫 번째 숫자가 1이면, 페티야는 문제의 해결책에 대해 확신하고 있는 것이고, 그렇지 않으면 확신하지 않는 것입니다. 두 번째 숫자는 바샤의 해결책에 대한 견해를 나타내고, 세 번째 숫자는 토냐의 견해를 나타냅니다. 줄의 숫자는 공백으로 구분되어 있습니다.

## 출력

친구들이 대회에서 구현할 문제의 개수를 나타내는 정수 하나를 출력하시오.
