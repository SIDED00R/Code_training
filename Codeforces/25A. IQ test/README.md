## 문제 설명

Bob은 IQ 테스트를 통과하기 위해 준비하고 있습니다. 이 테스트에서 가장 빈번한 과제는 주어진 $n$개의 숫자 중에서 다른 숫자와 다르게 나타나는 숫자를 찾는 것입니다. Bob은 일반적으로 하나의 숫자가 다른 숫자들과 짝수성에서 다르다는 것을 관찰했습니다. Bob을 도와주세요 — 그의 답변을 확인하기 위해, 주어진 $n$개의 숫자 중에서 짝수성에서 다른 숫자를 찾는 프로그램이 필요합니다.

## 입력

첫 번째 줄에는 정수 $n$ ($3 \le n \le 100$)이 주어지며, 이는 문제의 숫자 개수를 나타냅니다. 두 번째 줄에는 $n$개의 공백으로 구분된 자연수가 주어지며, 이 숫자는 100을 초과하지 않습니다. 이 숫자들 중 정확히 하나는 다른 숫자들과 짝수성에서 다릅니다.

## 출력

짝수성과 다른 숫자의 인덱스를 출력하시오. 숫자는 입력 순서에 따라 1부터 번호가 매겨진다.
