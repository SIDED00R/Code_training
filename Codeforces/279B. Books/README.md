## 문제 설명

Valera가 여유 시간이 생기면 도서관에 가서 책을 읽습니다. 오늘 그는 읽을 수 있는 자유 시간이 $t$ 분이 있습니다. 그래서 Valera는 도서관에서 $n$ 권의 책을 가져왔고, 각 책을 읽는 데 필요한 시간을 평가했습니다. 책의 번호는 1부터 $n$까지의 정수로 매깁니다. Valera는 $i$ 번째 책을 읽는 데 $a_i$ 분이 필요합니다. Valera는 임의의 책 번호 $i$를 선택하고, 이 책부터 차례로 책을 읽기로 결정했습니다. 즉, 그는 먼저 $i$ 번 책을 읽고, 그 다음 $i + 1$ 번 책, 그 다음 $i + 2$ 번 책을 읽는 식입니다. 그는 자유 시간이 다 소진되거나 $n$ 번째 책을 다 읽을 때까지 이 과정을 계속합니다. Valera는 각 책을 끝까지 읽으며, 읽을 시간이 부족한 경우에는 책을 읽기 시작하지 않습니다. Valera가 읽을 수 있는 최대 책의 수를 출력하세요.

## 입력

첫 번째 줄에는 두 개의 정수 $n$과 $t$가 포함되어 있습니다 ($1 \le n \le 10^5$; $1 \le t \le 10^9$) — 책의 수와 발레라가 가진 자유 시간(분)입니다. 두 번째 줄에는 $n$개의 정수 $a_1, a_2, \ldots, a_n$의 수열이 포함되어 있습니다 ($1 \le a_i \le 10^4$), 여기서 숫자 $a_i$는 소년이 $i$번째 책을 읽는 데 필요한 시간을 나타냅니다.

## 출력

단일 정수를 출력하시오 — 발레라가 읽을 수 있는 최대 책의 수.
