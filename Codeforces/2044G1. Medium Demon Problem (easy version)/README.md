## 문제 설명

이 문제는 쉬운 버전입니다. 두 버전 간의 주요 차이점은 굵게 표시되어 있습니다. $n$ 마리의 거미가 모여서 봉제 인형을 교환합니다. 처음에 각 거미는 $1$개의 봉제 인형을 가지고 있습니다. 매년, 거미 $i$가 봉제 인형을 최소한 하나 가지고 있다면, 그는 정확히 하나의 봉제 인형을 거미 $r_i$에게 줄 것입니다. 그렇지 않으면 그는 아무것도 하지 않습니다. 모든 봉제 인형의 이동은 동시에 발생한다는 점에 유의하세요. 이 버전에서는 어떤 거미라도 어느 시점에서 $1$개 이상의 봉제 인형을 가지고 있다면, 그들은 $1$개를 제외한 모든 것을 버립니다. 현재 연도의 과정이 안정적이려면 각 거미가 이전 연도(이전 연도의 교환 전)와 같은 수의 봉제 인형을 가져야 합니다. 연도 $1$은 절대 안정적일 수 없다는 점에 유의하세요. 과정이 안정적으로 되는 첫 번째 연도를 찾으세요.

## 입력

첫 번째 줄에는 정수 $t$ ($1 \leq t \leq 10^4$) 가 포함되어 있습니다 — 테스트 케이스의 수입니다. 각 테스트 케이스의 첫 번째 줄에는 정수 $n$ ($2 \leq n \leq 2 \cdot 10^5$) 가 포함되어 있습니다 — 거미의 수입니다. 다음 줄에는 $n$개의 정수 $r_1, r_2, \ldots, r_n$ ($1 \leq r_i \leq n, r_i \neq i$) 가 포함되어 있습니다 — 각 거미의 봉제 인형 수령자입니다. 모든 테스트 케이스에 대한 $n$의 합은 $2 \cdot 10^5$를 초과하지 않는 것이 보장됩니다.

## 출력

각 테스트 케이스에 대해, 프로세스가 안정화되는 첫 번째 연도를 새로운 줄에 정수로 출력하세요.
