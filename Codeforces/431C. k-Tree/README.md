## 문제 설명

최근에 창의적인 학생 레샤는 나무에 대한 강의를 들었습니다. 강의 후 레샤는 영감을 받아 자신만의 나무를 고안했으며, 이를 $k$-트리라고 부릅니다. $k$-트리는 다음과 같은 특성을 가진 무한 루트 트리입니다: 각 정점은 정확히 $k$개의 자식을 가지고 있으며, 각 간선은 어떤 가중치를 가집니다. 어떤 정점에서 자식으로 가는 간선(정확히 $k$개의 간선)을 살펴보면, 그 가중치는 $1, 2, 3, \ldots, k$와 같습니다. 아래 그림은 3-트리의 일부를 보여줍니다.

레샤의 좋은 친구 디마는 이 나무에 대해 알게 되자마자 궁금해졌습니다: "루트에서 시작하여 총 가중치 $n$(경로의 모든 간선 가중치의 합)을 가지며, 가중치가 최소 $d$인 간선이 적어도 하나 포함된 경로는 몇 개나 될까?" 디마의 질문에 대한 답을 찾는 데 도움을 주세요. 경우의 수가 상당히 클 수 있으므로, 결과를 $1000000007$ $(10^9 + 7)$로 나눈 나머지를 출력하세요.

## 입력

한 줄에는 세 개의 공백으로 구분된 정수 $n$, $k$, $d$가 포함되어 있습니다: $1 \le n, k \le 100; \; 1 \le d \le k$.

## 출력

단일 정수를 출력하시오 — 문제의 답을 $1000000007$ (즉, $10^9 + 7$)로 나눈 나머지로.
