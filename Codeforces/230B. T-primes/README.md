## 문제 설명

소수는 정확히 두 개의 서로 다른 양의 약수를 가진 양의 정수라는 것을 알고 있습니다. 이와 유사하게, 양의 정수 $t$가 정확히 세 개의 서로 다른 양의 약수를 가지면 $t$를 Т-소수라고 부르겠습니다. $n$개의 양의 정수로 이루어진 배열이 주어집니다. 각 정수가 Т-소수인지 여부를 판단하세요.

## 입력

첫 번째 줄에는 하나의 양의 정수 $n$ $(1 \le n \le 10^5)$가 포함되어 있으며, 이는 배열에 있는 숫자의 개수를 나타냅니다. 다음 줄에는 $n$개의 공백으로 구분된 정수 $x_i$ $(1 \le x_i \le 10^{12})$가 포함되어 있습니다. C++에서 64비트 정수를 읽거나 쓸 때는 %lld 지정자를 사용하지 마십시오. 대신 $cin$, $cout$ 스트림이나 %I64d 지정자를 사용하는 것이 좋습니다.

## 출력

$n$개의 줄을 출력하시오: $i$번째 줄에는 숫자 $x_i$가 T-프라임이면 "YES" (따옴표 없이)를, 그렇지 않으면 "NO" (따옴표 없이)를 포함해야 한다.
