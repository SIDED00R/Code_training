## 문제 설명

초보 프로그래머인 제니아는 $2n$개의 비음이 아닌 정수로 구성된 수열 $a$를 가지고 있습니다: $a_1, a_2, \ldots, a_{2n}$. 제니아는 현재 비트 연산을 공부하고 있습니다. 비트 연산이 어떻게 작동하는지 더 잘 이해하기 위해, 제니아는 수열 $a$에 대한 어떤 값 $v$를 계산하기로 결정했습니다. 즉, 값 $v$를 계산하기 위해 여러 번의 반복이 필요합니다. 첫 번째 반복에서 제니아는 새로운 수열 $a_1 \text{ or } a_2, a_3 \text{ or } a_4, \ldots, a_{2n-1} \text{ or } a_{2n}$을 작성하며, 이 수열은 $2n-1$개의 요소로 구성됩니다. 다시 말해, 그녀는 수열 $a$의 인접한 요소들의 비트 단위 OR을 기록합니다. 두 번째 반복에서는 첫 번째 반복 후에 얻은 수열의 인접한 요소들의 비트 단위 배타적 OR을 기록합니다. 세 번째 반복에서는 두 번째 반복 후에 얻은 수열의 인접한 요소들의 비트 단위 OR을 기록합니다. 이렇게 계속 진행되며, 비트 단위 배타적 OR과 비트 단위 OR이 번갈아 가며 수행됩니다. 결국, 그녀는 하나의 요소로 구성된 수열을 얻고, 그 요소가 바로 $v$입니다. 예를 들어 보겠습니다. 수열 $a = (1, 2, 3, 4)$라고 가정해봅시다. 그러면 모든 변환을 기록해 보겠습니다: $(1, 2, 3, 4) \rightarrow (1 \text{ or } 2 = 3, 3 \text{ or } 4 = 7) \rightarrow (3 \text{ xor } 7 = 4)$. 결과는 $v = 4$입니다. 제니아의 초기 수열이 주어집니다. 그러나 주어진 수열에 대해 값 $v$를 계산하는 것은 너무 쉬울 것이므로, 추가로 $m$개의 쿼리가 주어집니다. 각 쿼리는 정수 쌍 $p, b$로 구성됩니다. 쿼리 $p, b$는 $a_p = b$를 수행해야 함을 의미합니다. 각 쿼리 후에는 새로운 수열 $a$에 대한 새로운 값 $v$를 출력해야 합니다.

## 입력

첫 번째 줄에는 두 개의 정수 $n$과 $m$이 포함되어 있습니다 ($1 \le n \le 17$, $1 \le m \le 10^5$). 다음 줄에는 $2^n$개의 정수 $a_1, a_2, \ldots, a_{2^n}$이 포함되어 있습니다 ($0 \le a_i < 2^{30}$). 다음 $m$줄에는 쿼리가 포함되어 있습니다. $i$-번째 줄에는 정수 $p_i$, $b_i$가 포함되어 있습니다 ($1 \le p_i \le 2^n$, $0 \le b_i < 2^{30}$) — $i$-번째 쿼리입니다.

## 출력

$m$개의 정수를 출력하시오. $i$번째 정수는 $i$번째 쿼리 이후의 수열 $a$에 대한 값 $v$를 나타냅니다.
