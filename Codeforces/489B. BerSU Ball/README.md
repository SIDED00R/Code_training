## 문제 설명

베를란드 주립대학교는 100500주년 기념으로 무도회를 개최합니다! $n$명의 소년과 $m$명의 소녀가 이미 왈츠, 미뉴엣, 폴로네즈 및 쿼드릴 동작을 연습하고 있습니다. 우리는 여러 소년-소녀 쌍이 무도회에 초대될 것이라는 것을 알고 있습니다. 그러나 각 쌍의 파트너의 춤 실력은 최대 한 차이 나야 합니다. 각 소년의 춤 실력을 알고 있으며, 마찬가지로 각 소녀의 춤 실력도 알고 있습니다. $n$명의 소년과 $m$명의 소녀로부터 형성할 수 있는 최대 쌍의 수를 결정할 수 있는 코드를 작성하세요.

## 입력

첫 번째 줄에는 정수 $n$ $(1 \le n \le 100)$이 주어지며, 이는 소년의 수를 나타냅니다. 두 번째 줄에는 수열 $a_1, a_2, \ldots, a_n$ $(1 \le a_i \le 100)$이 주어지며, 여기서 $a_i$는 $i$번째 소년의 춤 실력을 나타냅니다. 마찬가지로, 세 번째 줄에는 정수 $m$ $(1 \le m \le 100)$이 주어지며, 이는 소녀의 수를 나타냅니다. 네 번째 줄에는 수열 $b_1, b_2, \ldots, b_m$ $(1 \le b_j \le 100)$이 주어지며, 여기서 $b_j$는 $j$번째 소녀의 춤 실력을 나타냅니다.

## 출력

단일 숫자를 출력하시오 — 필요한 최대 가능한 쌍의 수.
