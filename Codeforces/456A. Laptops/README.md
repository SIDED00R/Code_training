## 문제 설명

어느 날 디마와 알렉스는 노트북의 가격과 품질에 대해 논쟁을 벌였습니다. 디마는 노트북이 비쌀수록 더 좋다고 생각합니다. 알렉스는 이에 동의하지 않습니다. 알렉스는 첫 번째 노트북의 가격이 두 번째 노트북의 가격보다 작고(엄격히 작음), 첫 번째 노트북의 품질이 두 번째 노트북의 품질보다 높다고(엄격히 큼) 생각합니다. 알렉스의 추측이 맞는지 확인해 주세요. $n$개의 노트북에 대한 설명이 주어집니다. 위에서 설명한 두 개의 노트북이 존재하는지 여부를 판단하세요.

## 입력

첫 번째 줄에는 정수 $n$ $(1 \le n \le 10^5)$이 주어지며, 이는 노트북의 개수를 나타냅니다. 다음 $n$줄에는 각각 두 개의 정수 $a_i$와 $b_i$ $(1 \le a_i, b_i \le n)$가 포함되어 있습니다. 여기서 $a_i$는 $i$번째 노트북의 가격을 나타내고, $b_i$는 $i$번째 노트북의 품질을 나타내는 숫자입니다 (숫자가 클수록 품질이 높습니다). 모든 $a_i$는 서로 다릅니다. 모든 $b_i$도 서로 다릅니다.

## 출력

알렉스가 맞다면 " Happy Alex "를 출력하고, 그렇지 않다면 " Poor Alex "를 출력하세요 (따옴표 없이).
