## 문제 설명

A + B는 종종 어떤 대회 플랫폼을 보여주기 위해 가장 쉬운 문제의 예로 사용됩니다. 그러나 일부 과학자들은 때때로 이 문제가 수용되는 것이 그렇게 쉽지 않다는 것을 관찰했습니다. 도전해 보시겠습니까?

## 입력

입력에는 두 개의 정수 $a$와 $b$가 포함되어 있으며, ( $0 \leq a, b \leq 10^3$ ) 한 개의 공백으로 구분되어 있습니다.

## 출력

주어진 정수의 합을 출력하시오.
