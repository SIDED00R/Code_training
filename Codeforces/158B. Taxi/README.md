## 문제 설명

수업이 끝난 후, n개의 학교 친구 그룹이 밖으로 나가 폴리카르푸스를 방문하여 그의 생일을 축하하기로 결정했습니다. 우리는 i번째 그룹이 s_i명의 친구로 구성되어 있다는 것을 알고 있습니다 (1 ≤ s_i ≤ 4) 그리고 그들은 함께 폴리카르푸스에게 가고 싶어합니다. 그들은 택시를 타고 가기로 결정했습니다. 각 차는 최대 4명의 승객을 수송할 수 있습니다. 모든 그룹의 구성원이 같은 택시에 타야 한다면, 아이들이 필요한 최소한의 택시 수는 얼마일까요?

## 입력

첫 번째 줄에는 정수  n  ( 1 ≤  n  ≤ 10 5 ) — 학교 어린이 그룹의 수가 포함되어 있습니다. 두 번째 줄에는 정수의 시퀀스  s 1 ,  s 2 , ...,  s n  ( 1 ≤  s i  ≤ 4 )가 포함되어 있습니다. 정수는 공백으로 구분되어 있으며,  s i 는 i번째 그룹의 어린이 수입니다.

## 출력

모든 아이들을 폴리카르푸스에게 데려가기 위해 필요한 최소 택시 수를 출력하시오.
