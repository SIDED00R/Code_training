## 문제 설명

Ayush와 Ashish는 $n$개의 노드로 구성된 루트가 없는 트리에서 게임을 합니다. 노드는 $1$부터 $n$까지 번호가 매겨져 있습니다. 플레이어들은 다음과 같은 방식으로 차례로 이동합니다: 트리에서 임의의 리프 노드를 선택하고 이 노드가 하나의 끝점인 엣지와 함께 제거합니다. 리프 노드는 차수가 $1$ 이하인 노드입니다. 트리는 사이클이 없는 연결된 무방향 그래프입니다. 특별한 노드가 $x$라는 번호로 존재합니다. 이 노드를 제거하는 플레이어가 게임에서 승리합니다. Ayush가 먼저 이동합니다. 각 플레이어가 최적으로 플레이할 경우 게임의 승자를 결정하세요.

## 입력

입력의 첫 번째 줄에는 단일 정수 $t$ $(1 \leq t \leq 10)$가 포함되어 있으며, 이는 테스트 케이스의 수를 나타냅니다. 테스트 케이스의 설명이 이어집니다. 각 테스트 케이스의 첫 번째 줄에는 두 개의 정수 $n$과 $x$ $(1 \leq n \leq 1000, 1 \leq x \leq n)$가 포함되어 있으며, 이는 각각 트리의 노드 수와 특별한 노드를 나타냅니다. 다음 $n-1$개의 줄에는 두 개의 정수 $u$, $v$ $(1 \leq u, v \leq n, \text{ } u \ne v)$가 포함되어 있으며, 이는 트리에서 노드 $u$와 $v$ 사이에 엣지가 있음을 의미합니다.

## 출력

각 테스트 케이스에 대해, Ayush가 게임에서 이기면 " Ayush "를 출력하고, 그렇지 않으면 " Ashish "를 출력합니다 (따옴표 없이).
