## 문제 설명

주어진 배열 $a_1, a_2, \dots , a_n$은 $0$부터 $9$까지의 정수로 구성되어 있습니다. 부분 배열 $a_l, a_{l+1}, a_{l+2}, \dots , a_{r-1}, a_r$는 이 부분 배열의 요소의 합이 이 부분 배열의 길이와 같을 때 좋다고 합니다 ($\sum\limits_{i=l}^{r} a_i = r - l + 1$). 예를 들어, 만약 $a = [1, 2, 0]$라면, 좋은 부분 배열은 $3$개가 있습니다: $a_{1 \dots 1} = [1]$, $a_{2 \dots 3} = [2, 0]$ 그리고 $a_{1 \dots 3} = [1, 2, 0]$입니다. 배열 $a$의 좋은 부분 배열의 개수를 계산하세요.

## 입력

첫 번째 줄에는 하나의 정수 $t$ ($1 \le t \le 1000$) 가 포함되어 있으며, 이는 테스트 케이스의 수를 나타냅니다. 각 테스트 케이스의 첫 번째 줄에는 하나의 정수 $n$ ($1 \le n \le 10^5$) 가 포함되어 있으며, 이는 배열 $a$의 길이를 나타냅니다. 각 테스트 케이스의 두 번째 줄에는 $n$개의 십진수로 구성된 문자열이 포함되어 있으며, 여기서 $i$번째 숫자는 $a_i$의 값을 나타냅니다. 모든 테스트 케이스에 대한 $n$의 합이 $10^5$를 초과하지 않는다는 것이 보장됩니다.

## 출력

각 테스트 케이스에 대해 정수 하나를 출력하시오 — 배열 $a$의 좋은 부분 배열의 개수.
