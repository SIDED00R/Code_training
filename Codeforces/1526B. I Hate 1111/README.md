## 문제 설명

정수 $x$가 주어집니다. $11, 111, 1111, 11111, \ldots$ 중 일부를 더하여 $x$를 만들 수 있습니까? (이들 중 어떤 숫자도 원하는 만큼 사용할 수 있습니다). 예를 들어, $$33=11+11+11$$ $$144=111+11+11+11$$

## 입력

입력의 첫 번째 줄에는 단일 정수 $t$ $(1 \leq t \leq 10000)$가 포함되어 있습니다 — 테스트 케이스의 수입니다. 각 테스트 케이스의 첫 번째이자 유일한 줄에는 단일 정수 $x$ $(1 \leq x \leq 10^9)$가 포함되어 있습니다 — 만들어야 하는 숫자입니다.

## 출력

각 테스트 케이스에 대해, 단일 문자열을 출력해야 합니다. 만약 $x$를 만들 수 있다면 "YES" (따옴표 없이)를 출력하세요. 그렇지 않으면 "NO"를 출력하세요. "YES"와 "NO"의 각 글자는 대문자 또는 소문자 모두 사용할 수 있습니다.
