## 문제 설명

고양이는 pspspsps에 끌리지만, 위엄 있는 드래곤인 Evirir는 매우 특정한 요구 사항이 있는 pspspsps에만 끌립니다... 길이 $n$인 문자열 $s = s_1s_2\ldots s_n$이 문자 p, s, 및 . (점)으로 구성되어 있을 때, 모든 정수 $i$ ($1 \le i \le n$)에 대해 다음을 만족하는 길이 $n$의 순열$^{\text{∗}}$ $p$가 존재하는지 판단하세요:     만약 $s_i$가 p라면, $[p_1, p_2, \ldots, p_i]$는 길이 $i$의 순열을 형성해야 합니다;   만약 $s_i$가 s라면, $[p_i, p_{i+1}, \ldots, p_{n}]$는 길이 $n-i+1$의 순열을 형성해야 합니다;   만약 $s_i$가 .이라면, 추가적인 제한이 없습니다.  $^{\text{∗}}$길이 $n$의 순열은 $1$부터 $n$까지의 $n$개의 서로 다른 정수로 구성된 배열로, 임의의 순서로 나열된 것입니다. 예를 들어, $[2,3,1,5,4]$는 순열이지만, $[1,2,2]$는 순열이 아닙니다 ($2$가 배열에 두 번 나타나기 때문), 그리고 $[1,3,4]$도 순열이 아닙니다 ($n=3$인데 배열에 $4$가 있기 때문).

## 입력

각 테스트는 여러 개의 테스트 케이스로 구성됩니다. 첫 번째 줄에는 테스트 케이스의 개수 $t$ ($1 \le t \le 10^4$)가 포함되어 있습니다. 테스트 케이스의 설명이 이어집니다. 각 테스트 케이스의 첫 번째 줄에는 문자열 $s$의 길이인 정수 $n$ ($1 \le n \le 500$)이 포함되어 있습니다. 각 테스트 케이스의 두 번째 줄에는 문자 p, s, .로 구성된 길이 $n$의 문자열 $s$가 포함되어 있습니다. 모든 테스트 케이스에 대한 $n$의 합은 $5000$을 초과하지 않는 것이 보장됩니다.

## 출력

각 테스트 케이스에 대해 한 줄에  YES  또는  NO  를 출력하세요. 그러한 순열이 존재하면  YES  를 출력하고, 그렇지 않으면  NO  를 출력하세요. 대소문자에 관계없이 답변을 출력할 수 있습니다 (대문자 또는 소문자). 예를 들어, 문자열 " yEs ", " yes ", " Yes ", " YES "는 긍정적인 응답으로 인식됩니다.
