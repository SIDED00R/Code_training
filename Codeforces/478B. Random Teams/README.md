## 문제 설명

$n$명의 참가자들이 어떤 방식으로 $m$개의 팀으로 나뉘어져 각 팀에 최소한 한 명의 참가자가 있도록 하였습니다. 대회가 끝난 후, 같은 팀의 참가자들 간의 모든 쌍이 친구가 되었습니다. 당신의 임무는 대회가 끝날 무렵 형성될 수 있는 친구 쌍의 최소 개수와 최대 개수를 찾는 프로그램을 작성하는 것입니다.

## 입력

입력의 유일한 줄에는 두 개의 정수 $n$과 $m$이 공백 하나로 구분되어 있습니다 ($1 \le m \le n \le 10^9$) — 각각 참가자의 수와 팀의 수를 나타냅니다.

## 출력

출력의 유일한 줄에는 두 개의 정수 $k_{min}$과 $k_{max}$가 포함되어야 하며, 각각 친구 쌍의 최소 가능한 수와 최대 가능한 수를 나타냅니다.
