## 문제 설명

아기 에합은 특정 연산에 대한 사랑으로 유명합니다. 그는 길이가 $n$인 배열 $a$를 가지고 있으며, 다음과 같은 연산을 계속하기로 결정했습니다: 그는 $2$개의 인접한 요소를 선택한 후, 그것들을 제거하고 그 자리에 하나의 정수를 넣습니다: 그것들의 비트wise XOR입니다. 배열의 길이는 하나 줄어든다는 점에 유의하세요. 이제 그는 배열의 모든 요소를 같게 만들 수 있는지 묻습니다. 아기들이 당신의 삶을 더 어렵게 만들고 싶어하기 때문에, 그는 최소한 $2$개의 요소가 남아 있어야 한다고 요구합니다.

## 입력

첫 번째 줄에는 정수 $t$ ($1 \le t \le 15$) 가 포함되어 있습니다. 이는 해결해야 할 테스트 케이스의 수입니다. 각 테스트 케이스의 첫 번째 줄에는 정수 $n$ ($2 \le n \le 2000$) 이 포함되어 있습니다. 이는 배열 $a$의 요소 수입니다. 두 번째 줄에는 $n$개의 공백으로 구분된 정수 $a_1$, $a_2$, $\ldots$, $a_{n}$ ($0 \le a_i < 2^{30}$) 가 포함되어 있습니다. 이는 배열 $a$의 요소들입니다.

## 출력

아기 에합이 모든 요소를 같게 만들 수 있으면서 최소한 $2$개의 요소는 남겨둘 수 있다면 "YES"를 출력하세요. 그렇지 않다면 "NO"를 출력하세요.
