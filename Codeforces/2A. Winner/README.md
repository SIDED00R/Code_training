## 문제 설명

카드 게임 "Berlogging"에서의 승자는 다음 규칙에 따라 결정됩니다. 게임이 끝날 때 최대 점수를 가진 플레이어가 한 명뿐이라면, 그가 승자입니다. 하지만 그러한 플레이어가 두 명 이상일 경우 상황은 더 복잡해집니다. 각 라운드 동안 플레이어는 특정 점수를 얻거나 잃습니다. 게임 진행 중 점수는 "이름 점수" 형식으로 기록되며, 여기서  이름 은 플레이어의 이름이고,  점수 는 이번 라운드에서 얻은 점수로 정수입니다. 만약 점수가 음수라면, 이는 플레이어가 해당 라운드에서 잃었다는 것을 의미합니다. 따라서 게임이 끝날 때 두 명 이상의 플레이어가 최대 점수(예를 들어, $m$)를 가지고 있다면, 그들 중에서 먼저 최소 $m$ 점수를 기록한 플레이어가 승리합니다. 처음에는 각 플레이어가 0 점을 가지고 있습니다. 게임이 끝날 때 최소 한 명의 플레이어가 양의 점수를 가지고 있다는 것이 보장됩니다.

## 입력

첫 번째 줄에는 정수 $n$이 포함되어 있습니다 ($1 \le n \le 1000$). $n$은 플레이된 라운드의 수입니다. 그 다음에는 $n$개의 줄이 이어지며, 각 줄에는 "이름 점수" 형식으로 라운드에 대한 정보가 시간 순서대로 포함되어 있습니다. 여기서 $이름$은 길이가 1에서 32 사이인 소문자 라틴 문자로 이루어진 문자열이며, $점수$는 -1000에서 1000 사이의 정수입니다(포함).
