## 문제 설명

Bajtek는 아이스 스케이팅을 배우고 있습니다. 그는 초보자이기 때문에 그의 유일한 이동 방법은 북쪽, 동쪽, 남쪽 또는 서쪽으로 눈더미를 밀고 나가서 다른 눈더미에 착지할 때까지 미끄러지는 것입니다. 그는 이러한 방식으로 어떤 눈더미에서 다른 눈더미로 이동할 수 없는 경우가 있다는 것을 알아차렸습니다. 이제 그는 모든 눈더미에서 다른 눈더미로 이동할 수 있도록 추가적인 눈더미를 쌓고 싶어합니다. 그는 최소한으로 만들어야 할 눈더미의 개수를 찾도록 요청했습니다. 우리는 Bajtek가 정수 좌표에서만 눈더미를 쌓을 수 있다고 가정합니다.

## 입력

입력의 첫 번째 줄에는 하나의 정수 $n$ $(1 \le n \le 100)$이 주어지며, 이는 눈더미의 개수를 나타냅니다. 다음 $n$개의 줄에는 두 개의 정수 $x_i$와 $y_i$ $(1 \le x_i, y_i \le 1000)$가 포함되어 있으며, 이는 $i$번째 눈더미의 좌표를 나타냅니다. 북쪽 방향은 $Oy$ 축의 방향과 일치하므로, 동쪽 방향은 $Ox$ 축의 방향과 일치합니다. 모든 눈더미의 위치는 서로 다릅니다.

## 출력

바이텍이 어떤 눈더미에서 다른 눈더미로 갈 수 있도록 하기 위해 생성해야 하는 최소한의 눈더미 개수를 출력하시오.
