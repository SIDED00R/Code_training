## 문제 설명

점수는 $k$-위에 해당하는 참가자의 점수와 같거나 그보다 높은 점수를 얻은 참가자는 다음 라운드로 진출할 수 있습니다. 단, 참가자는 양의 점수를 얻어야 합니다... — 대회 규칙의 발췌문. 총 $n$명의 참가자가 대회에 참가하였고 ($n \ge k$), 그들의 점수를 이미 알고 있습니다. 다음 라운드로 진출할 참가자의 수를 계산하세요.

## 입력

입력의 첫 번째 줄에는 두 개의 정수 $n$과 $k$ ($1 \leq k \leq n \leq 50$)이 공백으로 구분되어 있습니다. 두 번째 줄에는 $n$개의 공백으로 구분된 정수 $a_1, a_2, \ldots, a_n$ ($0 \leq a_i \leq 100$)이 포함되어 있으며, 여기서 $a_i$는 $i$번째로 입상한 참가자가 얻은 점수입니다. 주어진 수열은 비증가 순서입니다 (즉, 모든 $i$에 대해 $1$부터 $n-1$까지 다음 조건이 만족됩니다: $a_i \geq a_{i+1}$).

## 출력

다음 라운드로 진출하는 참가자의 수를 출력하시오.
