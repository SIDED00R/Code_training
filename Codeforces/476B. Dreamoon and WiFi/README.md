## 문제 설명

드리무는 숫자선의 위치 $0$에 서 있습니다. 드라질은 드리무의 스마트폰으로 Wi-Fi를 통해 명령 목록을 보내고 드리무는 이를 따릅니다. 각 명령은 다음 두 가지 유형 중 하나입니다: 긍정 방향으로 1 단위 이동하는 명령은 '+'로 표시되고, 부정 방향으로 1 단위 이동하는 명령은 '-'로 표시됩니다. 그러나 Wi-Fi 상태가 너무 나빠서 드리무의 스마트폰은 일부 명령을 인식하지 못하고, 드리무는 인식된 명령 중 일부가 잘못되었을 수도 있다는 것을 알고 있습니다. 드리무는 인식된 모든 명령을 따르기로 결정하고 인식되지 않은 명령에 대해서는 공정한 동전을 던져 결정합니다 (즉, 그는 같은 확률 $0.5$로 부정 또는 긍정 방향으로 1 단위 이동합니다). 드라질이 보낸 원래 명령 목록과 드리무가 받은 목록이 주어졌을 때, 드리무가 드라질의 명령에 의해 원래 최종 위치에 도달할 확률은 얼마입니까?

## 입력

첫 번째 줄에는 문자열 $s_1$이 포함되어 있습니다. 이는 Drazil이 Dreamoon에게 보내는 명령으로, 이 문자열은 { '+', '-' } 집합의 문자만으로 구성됩니다. 두 번째 줄에는 문자열 $s_2$가 포함되어 있습니다. 이는 Dreamoon의 스마트폰이 인식하는 명령으로, 이 문자열은 { '+', '-', '?' } 집합의 문자만으로 구성됩니다. '?'는 인식되지 않은 명령을 나타냅니다. 두 문자열의 길이는 같으며 $10$을 초과하지 않습니다.

## 출력

단일 실수를 출력하여 확률에 해당합니다. 답은 상대적 또는 절대적 오차가 $10^{-9}$를 초과하지 않으면 올바른 것으로 간주됩니다.
