## 문제 설명

폴리카프는 소셜 네트워크에서 시간을 보내는 것을 매우 좋아합니다. 그가 가장 좋아하는 네트워크의 채팅 목록 페이지는 친구에게 메시지를 보낼 때 그 친구의 채팅이 페이지의 가장 위로 올라오도록 만들어져 있습니다. 다른 채팅의 상대적인 순서는 변하지 않습니다. 만약 이전에 이 친구와의 채팅이 없었다면, 새로운 채팅이 단순히 목록의 맨 위에 삽입됩니다. 채팅 목록이 처음에는 비어 있다고 가정할 때, 폴리카프의 메시지 순서를 고려하여 모든 메시지가 처리된 후의 채팅 목록을 만들어 보세요. 친구가 폴리카프에게 메시지를 보낸 경우는 없다고 가정합니다.

## 입력

첫 번째 줄에는 정수 $n$ ($1 \le n \le 200000$)이 포함되어 있으며, 이는 폴리카르푸스의 메시지 수를 나타냅니다. 다음 $n$줄에는 메시지가 전송된 순서에 따라 메시지 수신자의 이름이 나열됩니다. 각 참가자의 이름은 길이가 최대 $10$인 비어 있지 않은 소문자 영어 문자로 이루어진 시퀀스입니다.

## 출력

Polycarp가 대화한 모든 수신자를 그들과의 채팅 순서에 따라 위에서 아래로 출력하세요.
