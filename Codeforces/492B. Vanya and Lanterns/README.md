## 문제 설명

Vanya는 길이 $l$인 직선 도로를 따라 늦은 밤에 걷고 있습니다. 이 도로는 $n$개의 랜턴에 의해 밝혀져 있습니다. 도로의 시작점은 $0$에 해당하고, 끝점은 $l$에 해당하는 좌표계 시스템을 고려합시다. 그러면 $i$번째 랜턴은 점 $a_i$에 위치해 있습니다. 랜턴은 그로부터 최대 거리 $d$ 이내의 모든 도로의 점을 비춥니다. 여기서 $d$는 모든 랜턴에 공통인 어떤 양수입니다. Vanya는 도로 전체를 밝히기 위해 랜턴이 가져야 할 최소 조명 반경 $d$가 얼마인지 궁금해합니다.

## 입력

첫 번째 줄에는 두 개의 정수 $n$, $l$ ($1 \le n \le 1000$, $1 \le l \le 10^9$)이 주어지며, 각각 랜턴의 개수와 거리의 길이를 나타냅니다. 다음 줄에는 $n$개의 정수 $a_i$ ($0 \le a_i \le l$)가 주어집니다. 여러 개의 랜턴이 같은 지점에 위치할 수 있습니다. 랜턴은 거리의 양 끝에 위치할 수 있습니다.

## 출력

전체 거리를 밝히기 위해 필요한 최소 조명 반경 $d$를 출력하시오. 답안은 절대 오차 또는 상대 오차가 $10^{-9}$를 초과하지 않으면 정답으로 간주된다.
