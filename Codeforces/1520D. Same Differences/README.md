## 문제 설명

주어진 배열 $a$는 $n$개의 정수로 구성되어 있습니다. $i < j$이며 $a_j - a_i = j - i$를 만족하는 인덱스 쌍 $(i, j)$의 개수를 세세요.

## 입력

첫 번째 줄에는 하나의 정수 $t$ ($1 \le t \le 10^4$)가 포함되어 있습니다. 그 다음에 $t$개의 테스트 케이스가 이어집니다. 각 테스트 케이스의 첫 번째 줄에는 하나의 정수 $n$ ($1 \le n \le 2 \cdot 10^5$)이 포함되어 있습니다. 각 테스트 케이스의 두 번째 줄에는 $n$개의 정수 $a_1, a_2, \ldots, a_n$ ($1 \le a_i \le n$)이 포함되어 있습니다 — 배열 $a$. 모든 테스트 케이스에 대한 $n$의 합은 $2 \cdot 10^5$를 초과하지 않는 것이 보장됩니다.

## 출력

각 테스트 케이스에 대해 $i < j$이고 $a_j - a_i = j - i$인 인덱스 쌍 $(i, j)$의 개수를 출력하시오.
