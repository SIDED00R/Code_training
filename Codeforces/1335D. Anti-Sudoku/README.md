## 문제 설명

주어진 것은 올바른 스도쿠 퍼즐의 해답입니다. 스도쿠가 무엇인지 모른다면, 여기에서 읽어볼 수 있습니다. 아래 그림은 올바른 스도쿠 해답을 보여줍니다: 블록은 굵은 검은색으로 테두리가 그려져 있습니다. 당신의 임무는 이 필드에서 최대 $9$개의 요소를 변경하는 것입니다 (즉, 어떤 $1 \le i, j \le 9$를 선택하고 위치 $(i, j)$의 숫자를 $[1; 9]$ 범위의 다른 숫자로 변경하는 것입니다) 그렇게 해서 안티-스도쿠를 만드는 것입니다. 안티-스도쿠는 $9 \times 9$ 필드로, 다음과 같은 조건을 만족합니다: 이 필드의 모든 숫자는 $[1; 9]$ 범위에 있어야 하며; 각 행에는 최소 두 개의 같은 요소가 포함되어야 하고; 각 열에는 최소 두 개의 같은 요소가 포함되어야 하며; 각 $3 \times 3$ 블록(블록이 무엇인지 위의 링크를 통해 읽어볼 수 있습니다)에는 최소 두 개의 같은 요소가 포함되어야 합니다. 해답이 존재하는 것이 보장됩니다. 당신은 $t$개의 독립적인 테스트 케이스에 답해야 합니다.

## 입력

입력의 첫 번째 줄에는 하나의 정수 $t$ ($1 \le t \le 10^4$) 가 주어지며, 이는 테스트 케이스의 수를 나타냅니다. 그 다음에 $t$ 개의 테스트 케이스가 이어집니다. 각 테스트 케이스는 $9$ 줄로 구성되어 있으며, 각 줄은 공백 없이 $1$부터 $9$까지의 문자로 이루어진 스도쿠 퍼즐의 올바른 해답입니다.

## 출력

각 테스트 케이스에 대해 답을 출력하세요 — 최대 $9$개의 변경된 요소가 있는 초기 필드로, 얻어진 필드는 반 스도쿠가 되어야 합니다. 여러 솔루션이 있을 경우, 아무거나 출력해도 됩니다. 답이 존재하는 것이 보장됩니다.
