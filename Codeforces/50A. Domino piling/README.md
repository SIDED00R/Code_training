## 문제 설명

M × N 크기의 직사각형 보드가 주어집니다. 또한 2 × 1 크기의 표준 도미노 조각이 무한히 주어집니다. 조각을 회전시키는 것은 허용됩니다. 다음 조건을 만족하도록 보드에 가능한 한 많은 도미노를 배치해야 합니다: 1. 각 도미노는 두 개의 정사각형을 완전히 덮어야 합니다. 2. 두 개의 도미노가 겹치지 않아야 합니다. 3. 각 도미노는 보드 안에 완전히 있어야 하며, 보드의 가장자리에 닿는 것은 허용됩니다. 이러한 제한 조건 하에서 배치할 수 있는 도미노의 최대 개수를 구하세요.

## 입력

한 줄에 두 개의 정수 M과 N이 주어집니다 — 정사각형으로 이루어진 보드의 크기 (1 ≤ M ≤ N ≤ 16).

## 출력

하나의 숫자를 출력하시오 — 놓을 수 있는 최대 도미노의 수.
