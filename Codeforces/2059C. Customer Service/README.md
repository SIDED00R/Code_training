## 문제 설명

Nikyr는 "Black Contour" 회사에서 대기열 관리자로 일하기 시작했습니다. 그는 고객 서비스를 위한 순서를 선택해야 합니다. 총 $n$개의 대기열이 있으며, 각 대기열에는 처음에 $0$명의 사람이 있습니다. 다음 $n$개의 시간 순간 동안 두 가지 연속적인 사건이 발생합니다: 모든 대기열에 새로운 고객이 도착합니다. 보다 구체적으로, $j$번째 시간 순간에 $i$번째 대기열의 인원 수는 양의 정수 $a_{i,j}$만큼 증가합니다. Nikyr는 그 순간에 $n$개의 대기열 중에서 정확히 하나를 선택하여 서비스를 받도록 합니다. 이 대기열의 고객 수는 $0$이 됩니다. 모든 사건이 발생한 후 $i$번째 대기열의 인원 수를 $x_i$라고 할 때, Nikyr는 $x_1, x_2, \ldots, x_n$의 MEX$^{\dagger}$을 가능한 한 크게 하기를 원합니다. 최적의 대기열 서비스 순서를 통해 그가 달성할 수 있는 최대 값을 결정하는 데 도움을 주세요. $^{\dagger}$정수 집합 $c_1, c_2, \ldots, c_k$의 최소 제외(MEX)는 집합 $c$에 포함되지 않는 가장 작은 비음수 정수 $y$로 정의됩니다. 예를 들어: $\operatorname{MEX}([2,2,1])= 0$, 왜냐하면 $0$은 배열에 포함되지 않기 때문입니다. $\operatorname{MEX}([3,1,0,1]) = 2$, 왜냐하면 $0$과 $1$은 배열에 포함되지만 $2$는 포함되지 않기 때문입니다. $\operatorname{MEX}([0,3,1,2]) = 4$, 왜냐하면 $0$, $1$, $2$, $3$은 배열에 포함되지만 $4$는 포함되지 않기 때문입니다.

## 입력

각 테스트는 여러 개의 테스트 케이스로 구성됩니다. 첫 번째 줄에는 단일 정수 $t$ ($1 \le t \le 2 \cdot 10^4$) — 테스트 케이스의 수가 포함됩니다. 테스트 케이스의 설명이 이어집니다. 각 테스트 케이스의 첫 번째 줄에는 단일 정수 $n$ ($1 \le n \le 300$) — 대기열의 수와 시간의 순간이 포함됩니다. 다음 $n$개의 줄 중 $i$번째 줄에는 $n$개의 정수 $a_{i,1}, a_{i,2}, \ldots, a_{i,n}$ ($1 \le a_{i,j} \le 10^9$) — 각 시간의 $i$번째 대기열에 있는 새로운 고객의 수가 포함됩니다. 모든 테스트 케이스에 대해 $n^2$의 합이 $2 \cdot 10^5$를 초과하지 않도록 보장됩니다.

## 출력

각 테스트 케이스에 대해, 달성할 수 있는 최대 값 $\operatorname{MEX}([x_1, x_2, \ldots, x_n])$을 단일 정수로 출력하세요.
