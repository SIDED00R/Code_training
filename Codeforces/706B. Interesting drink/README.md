## 문제 설명

Vasiliy는 힘든 일을 한 후에 휴식을 취하는 것을 좋아하므로, 근처의 바에서 그를 자주 만날 수 있습니다. 모든 프로그래머들처럼, 그는 유명한 음료 "Beecola"를 좋아하며, 이 음료는 도시의 $n$ 개의 다른 가게에서 구매할 수 있습니다. 가게 $i$에서 한 병의 가격은 $x_i$ 동전으로 알려져 있습니다. Vasiliy는 $q$ 일 연속으로 그의 좋아하는 음료를 구매할 계획입니다. 그는 $i$번째 날에 $m_i$ 동전을 쓸 수 있다는 것을 알고 있습니다. 이제 그는 각 날마다 "Beecola"의 병을 구매할 수 있는 가게의 수가 몇 개인지 알고 싶어합니다.

## 입력

입력의 첫 번째 줄에는 단일 정수 $n$ $(1 \le n \le 100000)$이 주어지며, 이는 도시에서 바실리의 좋아하는 음료를 판매하는 상점의 수를 나타냅니다. 두 번째 줄에는 $n$개의 정수 $x_i$ $(1 \le x_i \le 100000)$가 주어지며, 이는 $i$번째 상점에서 음료의 병 가격을 나타냅니다. 세 번째 줄에는 단일 정수 $q$ $(1 \le q \le 100000)$가 주어지며, 이는 바실리가 음료를 구매할 계획인 날의 수를 나타냅니다. 그 다음에는 $q$줄이 이어지며, 각 줄에는 하나의 정수 $m_i$ $(1 \le m_i \le 10^9)$가 주어지며, 이는 바실리가 $i$번째 날에 사용할 수 있는 동전의 수를 나타냅니다.

## 출력

$q$개의 정수를 출력하세요. $i$번째 정수는 바실리가 $i$번째 날에 음료수 한 병을 구매할 수 있는 가게의 수와 같아야 합니다.
