## 문제 설명

프로그래머인 당신은 배열을 매우 좋아합니다. 당신의 생일을 맞아 친구들이 $n$개의 서로 다른 정수로 구성된 배열 $a$를 선물했습니다. 불행히도, 배열 $a$의 크기가 너무 작습니다. 당신은 더 큰 배열을 원합니다! 친구들은 당신이 더 큰 배열을 주겠다고 동의하지만, 단 하나의 조건이 있습니다: 배열 $a$를 (오름차순으로) 정렬할 수 있는지 여부를 정확히 대답해야 합니다. 즉, 배열 $a$의 정확히 하나의 구간을 뒤집어서 정렬할 수 있는지 확인해야 합니다. 구간과 뒤집기의 정의는 노트를 참조하세요.

## 입력

입력의 첫 번째 줄에는 정수 $n$ $(1 \le n \le 10^5)$이 포함되어 있으며, 이는 배열 $a$의 크기입니다. 두 번째 줄에는 $n$개의 서로 다른 공백으로 구분된 정수: $a[1], a[2], \ldots, a[n]$ $(1 \le a[i] \le 10^9)$가 포함되어 있습니다.

## 출력

"yes" 또는 "no" (따옴표 없이)를 출력하시오. 만약 당신의 대답이 "yes"라면, 또한 반전할 구간의 시작과 끝 (시작은 끝보다 크지 않아야 함) 인덱스를 나타내는 두 개의 공백으로 구분된 정수를 출력하시오. 이러한 인덱스를 선택하는 방법이 여러 가지인 경우, 그 중 아무 것이나 출력하시오.
