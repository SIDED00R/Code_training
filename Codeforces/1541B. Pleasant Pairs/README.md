## 문제 설명

주어진 배열 $a_1, a_2, \dots, a_n$는 $n$개의 서로 다른 정수로 구성되어 있습니다. $i < j$이고 $a_i \cdot a_j = i + j$인 인덱스 쌍 $(i, j)$의 개수를 세십시오.

## 입력

첫 번째 줄에는 하나의 정수 $t$ ($1 \leq t \leq 10^4$) 가 주어지며, 이는 테스트 케이스의 수입니다. 그 다음에 $t$ 개의 케이스가 이어집니다. 각 테스트 케이스의 첫 번째 줄에는 하나의 정수 $n$ ($2 \leq n \leq 10^5$) 가 주어지며, 이는 배열 $a$의 길이입니다. 각 테스트 케이스의 두 번째 줄에는 $n$개의 공백으로 구분된 정수 $a_1, a_2, \ldots, a_n$ ($1 \leq a_i \leq 2 \cdot n$) 가 주어지며, 이는 배열 $a$입니다. 모든 요소가 서로 다르다는 것이 보장됩니다. 모든 테스트 케이스에 대한 $n$의 합은 $2 \cdot 10^5$를 초과하지 않습니다.

## 출력

각 테스트 케이스에 대해, $i < j$이고 $a_i \cdot a_j = i + j$인 인덱스 쌍 $(i, j)$의 개수를 출력하세요.
