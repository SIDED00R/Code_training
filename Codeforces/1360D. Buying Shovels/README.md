## 문제 설명

폴리카르프는 정확히 $n$ 개의 삽을 사려고 합니다. 가게에서는 삽이 들어 있는 패키지를 판매합니다. 이 가게에는 $k$ 종류의 패키지가 있습니다: $i$번째 종류의 패키지는 정확히 $i$ 개의 삽으로 구성되어 있습니다 ($1 \le i \le k$). 가게에는 각 종류의 패키지가 무한히 있습니다. 폴리카르프는 하나의 패키지 종류를 선택한 다음, 이 종류의 패키지를 여러 개(하나 이상) 구매하고자 합니다. 폴리카르프가 정확히 $n$ 개의 삽을 얻기 위해 구매해야 하는 패키지의 최소 개수는 얼마일까요? 예를 들어, $n=8$이고 $k=7$인 경우, 폴리카르프는 $4$ 개의 삽이 들어 있는 패키지를 $2$ 개 구매할 것입니다. 폴리카르프가 다음 조건을 만족하면서 구매해야 하는 패키지의 최소 개수를 찾는 것을 도와주세요:     총 $n$ 개의 삽을 정확히 구매해야 하며;   구매할 모든 패키지의 크기는 동일하고, 각 패키지에 들어 있는 삽의 개수는 $1$부터 $k$까지의 정수입니다.

## 입력

첫 번째 줄에는 정수 $t$ ($1 \le t \le 100$) 가 포함되어 있으며, 이는 입력의 테스트 케이스 수를 나타냅니다. 그 다음으로 $t$개의 테스트 케이스가 각 줄마다 이어집니다. 각 테스트 케이스는 두 개의 양의 정수 $n$ ($1 \le n \le 10^9$)과 $k$ ($1 \le k \le 10^9$)로 구성되며, 이는 삽의 수와 패키지의 종류 수를 나타냅니다.

## 출력

$t$개의 테스트 케이스에 대한 답을 출력하시오. 각 답은 양의 정수로, 최소 패키지 수를 나타냅니다.
