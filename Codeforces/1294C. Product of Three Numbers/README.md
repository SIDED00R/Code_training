## 문제 설명

하나의 정수 $n$이 주어집니다. $2 \le a, b, c$이고 $a \cdot b \cdot c = n$인 서로 다른 정수 $a, b, c$를 찾거나, 불가능하다고 말하세요. 여러 개의 답이 있을 경우, 아무거나 출력해도 됩니다. $t$개의 독립적인 테스트 케이스에 대해 답해야 합니다.

## 입력

입력의 첫 번째 줄에는 하나의 정수 $t$ ($1 \le t \le 100$) 가 주어지며, 이는 테스트 케이스의 수를 나타냅니다. 다음 $n$ 줄은 테스트 케이스를 설명합니다. $i$번째 테스트 케이스는 새로운 줄에 정수 $n$ ($2 \le n \le 10^9$) 하나로 주어집니다.

## 출력

각 테스트 케이스에 대해 답을 출력하세요. 만약 $n$을 서로 다른 정수 $a, b, c$로 $a \cdot b \cdot c$ 형태로 표현하는 것이 불가능하다면 " NO "를 출력하세요. 그렇지 않으면 " YES "를 출력하고 가능한 표현 중 하나를 출력하세요.
