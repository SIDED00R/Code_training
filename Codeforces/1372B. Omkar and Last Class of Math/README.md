## 문제 설명

옴카르의 마지막 수학 수업에서 그는 최소 공배수, 즉 $LCM$에 대해 배웠습니다. $LCM(a, b)$는 $a$와 $b$ 모두로 나누어 떨어지는 가장 작은 양의 정수 $x$입니다. 호기심이 많은 옴카르는 즉시 $LCM$ 연산과 관련된 문제를 생각해냈습니다: 정수 $n$이 주어졌을 때, $a + b = n$을 만족하고 $LCM(a, b)$가 가능한 최소값이 되도록 하는 양의 정수 $a$와 $b$를 찾는 것입니다. 옴카르가 그의 매우 도전적인 수학 문제를 해결하는 데 도움을 줄 수 있나요?

## 입력

각 테스트는 여러 개의 테스트 케이스를 포함합니다. 첫 번째 줄에는 테스트 케이스의 수 $t$ ($1 \leq t \leq 10$)가 포함되어 있습니다. 테스트 케이스의 설명이 이어집니다. 각 테스트 케이스는 하나의 정수 $n$ ($2 \leq n \leq 10^{9}$)로 구성됩니다.

## 출력

각 테스트 케이스에 대해, 두 개의 양의 정수 $a$와 $b$를 출력하세요. 여기서 $a + b = n$이며 $LCM(a, b)$가 가능한 최소값이 되도록 해야 합니다.
