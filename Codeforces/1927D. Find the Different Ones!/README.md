## 문제 설명

주어진 배열 $a$는 $n$개의 정수로 구성되어 있으며, $q$개의 쿼리가 있습니다. 각 쿼리는 두 개의 정수 $l$과 $r$로 표현됩니다 ($1 \le l \le r \le n$). 여러분의 임무는 각 쿼리에 대해 두 개의 인덱스 $i$와 $j$를 찾는 것입니다 (또는 존재하지 않음을 판단하는 것입니다) 다음 조건을 만족해야 합니다:    $l \le i \le r$;   $l \le j \le r$;   $a_i \ne a_j$. 즉, 각 쿼리에 대해 $a_l, a_{l+1}, \dots, a_r$ 중에서 서로 다른 두 요소의 쌍을 찾아야 하며, 그러한 쌍이 존재하지 않음을 보고해야 합니다.

## 입력

입력의 첫 번째 줄에는 테스트 케이스의 수를 나타내는 정수 $t$ ($1 \le t \le 10^4$) 가 포함되어 있습니다. 테스트 케이스의 설명이 이어집니다. 각 테스트 케이스의 첫 번째 줄에는 배열 $a$의 길이를 나타내는 정수 $n$ ($2 \le n \le 2 \cdot 10^5$) 가 포함되어 있습니다. 각 테스트 케이스의 두 번째 줄에는 $n$개의 정수 $a_1, a_2, \dots, a_n$ ($1 \le a_i \le 10^6$) 가 포함되어 있습니다. 각 테스트 케이스의 세 번째 줄에는 쿼리의 수를 나타내는 정수 $q$ ($1 \le q \le 2 \cdot 10^5$) 가 포함되어 있습니다. 다음 $q$줄에는 각각 두 개의 정수 $l$과 $r$ ($1 \le l < r \le n$) 가 포함되어 있으며, 이는 쿼리의 경계를 나타냅니다. 모든 테스트 케이스에서 $n$의 값의 합은 $2 \cdot 10^5$를 초과하지 않으며, 마찬가지로 모든 테스트 케이스에서 $q$의 값의 합도 $2 \cdot 10^5$를 초과하지 않습니다.

## 출력

각 쿼리에 대해 공백으로 구분된 두 정수를 출력하십시오: $i$와 $j$ ($l \le i, j \le r$), 여기서 $a_i \ne a_j$입니다. 만약 그러한 쌍이 존재하지 않으면 $i=-1$과 $j=-1$을 출력하십시오. 테스트 케이스의 출력은 빈 줄로 구분할 수 있습니다. 이는 필수 요구 사항은 아닙니다.
